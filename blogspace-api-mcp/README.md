
## API Endpoints

- `GET /api/blogs` - Get all blogs
- `GET /api/blogs/:id` - Get blog by ID

## MCP Components

### Tools
- `get_all_blogs` - Fetch all blogs
- `get_blog_by_id` - Fetch specific blog

### Resources
- `blog://list` - List all blogs
- `blog://post/{id}` - Individual blog content

### Prompts
- `summarize_blog` - Summarize a blog post
- `compare_blogs` - Compare two blog posts

## Tech Stack

- Node.js
- Express
- @modelcontextprotocol/sdk
- ES Modules
